This folder contains all the code, report, image report and dataset for the assigned project of CVPR.
In order to run the code use main.py and select the task to run by activating the boolean variable associated to the desired task
and run the main.py file.
The code implementation for the tasks is inside the folder 'tasks'.

The pdf file CPVR_Crosariol_Matteo is the pdf report about the project implementation.

IMPORTANT!
In order to run the program it is needed to add the folder 'Dataset' at the same folder level of main.py. This folder is the datset of project assignment just renaimed as 'Dataset'.
